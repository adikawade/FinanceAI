# FinanceAI — Tax Engine

**Document Version:** 1.0.0

## Overview
FinanceAI includes a configurable, cloud-driven tax engine designed for Indian tax workflows.

## Features
- Tax Readiness Score
- Old vs New Regime Comparison
- Deduction Tracking (80C, 80D, etc.)
- ITR Preparation Assistant
- CA Export Package

## Design Principles
- No hardcoded tax rules
- Rules configurable from Firebase
- AI provides guidance only
- Final filing remains under user/CA control
