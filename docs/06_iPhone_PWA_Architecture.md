# FinanceAI — iPhone & Web PWA Architecture

**Document Version:** 1.0.0

## Overview
FinanceAI uses a Next.js Progressive Web App for iPhone and Web to provide a near-native experience.

## Core Technologies
- Next.js
- React
- TypeScript
- Tailwind CSS
- Service Workers
- Workbox

## Offline Strategy
- App Shell Cache
- Background Sync
- Stale-While-Revalidate
- IndexedDB Cache

## iOS Integration
- Apple Shortcuts
- Share Sheet Support
- Push Notifications
- Installable Home Screen Experience
